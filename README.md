# Jdbc::Altibase

This gem bundles the Altibase JDBC jar file for running in JRuby.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'jdbc-altibase'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install jdbc-altibase

## Usage

    require 'jdbc/altibase'
